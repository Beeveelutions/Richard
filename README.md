#Richard Read-Me

Program by Madeleine, Mads and Vee

Welcome to our program, Richard. 

Description:

This program simulates the Ricart-Agrawala algorythm in one go process using three local host clients with each their own GRPC server.

How to Use:

Run the client program from the Richard server with the command "go run Client/Client.go"

You can either control C to end the program or the program will auto terminate after 5 minutes.
