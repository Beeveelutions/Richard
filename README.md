# Richard Read-Me

Program by Madeleine, Mads and Vee

Welcome to our program, Richard. 

Description:

This program simulates the Ricart-Agrawala algorythm in 3 go process using three local host clients with each their own GRPC server.

How to Use:

Run the client program from the Richard server with the command "go run Client/Client.go"

You can either control C to end the program.

Run the client 3 times and enter 0, 1 and 2 into the respective terminals. (Code will not work if there isnt three clients present)
